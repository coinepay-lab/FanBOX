一、什么是 FanBOX

FanBOX 是由 Pixiv 推出的创作者赞助平台。

它类似于会员订阅平台，让粉丝可以长期支持创作者。

主要功能包括：

发布文章、图片和视频内容

粉丝订阅支持创作者

会员专属内容访问

平台通常会向创作者收取约 10% 的服务费用。

三、如何注册 FanBOX 账号

FanBOX 与 Pixiv 账号互通。

登录方式包括：

四、如何找到喜欢的创作者

登录 FanBOX 后，可以通过以下方式寻找创作者。

搜索创作者

在搜索框输入：

作者名字

作品名称

即可找到对应创作者。

浏览推荐创作者

点击：

发现创作者（Discover Creators）

可以查看热门创作者。

查看关注的 Pixiv 作者

如果你已经在 Pixiv 关注某位作者，通常可以在 FanBOX 找到他们的赞助页面。

五、选择支持档位

每个创作者通常会设置不同的赞助等级。

例如：

档位	月费	内容
基础档	¥300	基础更新内容
高级档	¥1000	独家作品
VIP档	¥3000	创作过程与特殊内容

在页面右侧可以查看不同档位的内容。

选择适合自己的支持等级即可。

六、FanBOX 支付步骤

完成档位选择后即可进行支付。

第一步：点击赞助

点击页面上的：

赞助（Support）

进入支付页面。

第二步：选择信用卡支付

选择：

Credit Card

作为支付方式。

第三步：填写虚拟信用卡信息

输入 CoinePay 提供的虚拟信用卡信息：

卡号

有效期

CVV

支付时通常会进行 约 0.02 美元的小额验证。

第四步：确认支付

确认金额后提交支付即可。

支付成功后即可访问创作者的会员内容。
